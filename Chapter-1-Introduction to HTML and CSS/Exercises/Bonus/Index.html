HTML & CSS Quick Reference Guide

HTML Tags

1. Paragraph (<p>)

The paragraph tag defines a block of text.

<p>This is a paragraph.</p>

2. Heading (<h1> - <h6>)

Headings define content sections, from most important (<h1>) to least (<h6>).

<h1>Main Heading</h1>
<h2>Subheading</h2>

3. Anchor (<a>)

The anchor tag creates hyperlinks.

<a href="https://www.example.com">Visit Example</a>

4. Image (<img>)

The image tag embeds images.

<img src="image.jpg" alt="Description">

5. Unordered List (<ul>)

Creates a bullet-point list.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

6. Ordered List (<ol>)

Creates a numbered list.

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

7. Table (<table>)

Defines a table.

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

8. Div (<div>)

Defines a section or container.

<div class="container">Content here</div>

9. Span (<span>)

Applies inline styling.

<p>This is a <span style="color: red;">red text</span>.</p>

10. Form (<form>)

Creates an input form.

<form>
    <input type="text" placeholder="Enter name">
    <button type="submit">Submit</button>
</form>

CSS Properties

1. Color

p {
    color: blue;
}

2. Background-color

body {
    background-color: #f0f0f0;
}

3. Font-size

h1 {
    font-size: 24px;
}

4. Margin

div {
    margin: 20px;
}

5. Padding

div {
    padding: 10px;
}

6. Border

table {
    border: 1px solid black;
}

7. Display

span {
    display: inline-block;
}

8. Text-align

h1 {
    text-align: center;
}
